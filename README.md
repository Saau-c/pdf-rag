# pdf-rag
A small PDF Q&amp;A (RAG) web app<br>
upload a PDF → we split it into chunks → index chunks with vector embeddings → retrieve relevant chunks for a question → (optionally) use an LLM to generate a concise answer with citations.<br><br>

<b>Prerequisites</b><br>
1. Must have python v3 or above<br>
2. In your terminal, go to your project folder and install all the required dependencies using: "pip install -r requirements.txt"<br><br>



<b>How to run:</b><br>
1.	GO to your project path<br>
2.  Create the virtual environment "venv" and activate it.<br>
3.	Run app.py: "streamlit run app.py"<br>
